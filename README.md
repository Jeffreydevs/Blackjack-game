Blackjack Game 🎰🃏

A simple and interactive Blackjack game built using HTML, CSS, and JavaScript.

🚀 Features:
Draw random cards
Track player score
Blackjack win detection
Dealer-style gameplay logic
Responsive UI
Restart / replay functionality

🛠️ Technologies Used:
HTML5
CSS3
JavaScript

🎮 How to Play
Start the game
Draw cards using the buttons
Try to reach 21
If you exceed 21, you lose
Hit Blackjack to win instantly
